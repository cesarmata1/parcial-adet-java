# Parcial ADET Java

Este repositorio contiene la solución desarrollada para un **parcial (examen parcial)** de la asignatura **ADET** usando **Java**. El proyecto incluye la implementación de la lógica necesaria para cumplir los requisitos planteados durante la evaluación.

## Descripción

El código está escrito en **Java** y resuelve un conjunto de ejercicios diseñados para evaluar los conocimientos adquiridos en la asignatura hasta ese momento. La práctica se centra en aplicar programación orientada a objetos, estructuras de control y lógica para ofrecer una solución funcional.

## Tecnologías utilizadas

- Java
- Entorno de desarrollo compatible con Java (por ejemplo, IntelliJ IDEA, Eclipse o NetBeans)
